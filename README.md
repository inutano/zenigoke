# zenigoke (project code name: kknmsmd)

Multiomics database for *Marchantia polymorpha* (zenigoke / 銭苔).

Phase 1 is processing 157 public SRA experiments (ChIP-seq, ATAC-seq,
Bisulfite-seq) against the MpTak v7.1 standard reference and curating
BioSample metadata with bsllmner-mk2 + Plant Ontology.

See `docs/superpowers/specs/` for the design document.
